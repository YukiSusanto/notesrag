# 深度学习笔记

## Transformer 架构

Transformer 的核心是自注意力机制（Self-Attention），它允许模型在处理序列时关注输入的所有位置，而不是像 RNN 那样只能顺序处理。

自注意力的计算分为三步：
1. 用输入向量分别乘以 W^Q、W^K、W^V 矩阵得到 Q、K、V
2. 计算注意力分数：Attention(Q,K,V) = softmax(QK^T/√d_k)V
3. 加权求和得到输出

Multi-Head Attention 将上述过程并行做多次，让模型同时关注不同子空间的信息。

## GPT 原理

GPT（Generative Pre-trained Transformer）是一个基于 Transformer Decoder 的自回归语言模型。它的核心思想是：给定前文，预测下一个 token。

训练过程分两个阶段：
- 预训练（Pre-training）：在海量文本上做语言建模
- 微调（Fine-tuning）：在特定任务数据上调整

## RAG 简介

RAG（Retrieval-Augmented Generation）将信息检索与文本生成结合：
1. 用户提问后，先从知识库检索相关文档
2. 将检索结果作为上下文拼入 prompt
3. LLM 基于检索到的知识生成回答

这种方式可以有效减少大模型的幻觉问题。
