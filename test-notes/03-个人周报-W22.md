## 学习笔记：RAG 评估指标

| 指标 | 含义 | 目标值 |
|------|------|--------|
| Recall@5 | 前 5 个检索结果中包含答案的比例 | > 90% |
| Faithfulness | 回答是否基于检索内容，无幻觉 | > 95% |
| Answer Relevance | 回答与问题的相关度 | > 85% |
| Latency P50 | 50% 请求的响应时间 | < 3s |
| Latency P99 | 99% 请求的响应时间 | < 10s |
